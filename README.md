# Cereal Nutrition Analysis

## Project Overview
This project performs exploratory data analysis (EDA), statistical analysis, and predictive modeling on cereal nutrition data using Python. The analysis focuses on identifying relationships between nutritional attributes, manufacturers, shelf placement, and consumer ratings.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Scikit-learn
- Jupyter Notebook

---

## Key Features
- Missing value treatment and outlier handling
- Exploratory data analysis and visualization
- Correlation and statistical hypothesis testing
- ANOVA and Chi-Square analysis
- Linear Regression model for rating prediction
- Data preprocessing using IQR and median replacement

---

## Statistical Analyses Performed
- Pearson Correlation Analysis
- ANOVA Testing
- Chi-Square Test of Independence
- Comparative Distribution Analysis
- Cross-tabulation Analysis

---

## Machine Learning
A Linear Regression model was developed using the top nutritional factors influencing cereal ratings. Model performance was evaluated using a 70:30 train-test split.
